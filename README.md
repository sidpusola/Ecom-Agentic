# Ecom-Agentic
ShopEase is a 24/7 agentic RAG chatbot for Indian e-commerce, built with an 8-node LangGraph pipeline, ChromaDB over 12 policy docs, DuckDuckGo fallback, Groq llama3.1-8b, and a self-reflection eval loop (retry if faithfulness &lt;0.7). Deployed on Streamlit. RAGAS: faithfulness ≥0.88, relevancy ≥0.86.
